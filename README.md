Course project for COMS 4156 - Advanced Software Engineering

## Instructions
To run, first clone this repository

To install requirements:
```sh
$pip install -r requirements.txt
```

To compile all tests:

```sh
$pytest
```

To deploy:
```sh
$python3 app.py
```
